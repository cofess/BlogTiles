Get started!

Posted by Admin on July 2015

In order to use BlogTiles template, you have to provide some images for the tiles. Check the HTML code from the index.html file and find an element with class of tl-page.

Each of these elements contain several divs that represent the tiles (This template uses the jsTiles plugin to create the tiles). Just place the path of your image as value to data-bt-bg attribute.

You can customize the appearance of the tiles via the template creation that jsTiles plugin provide.

Feel free to change any part of code you want in order to achieve a better result for you! Use this template for free to any commercial or not project.

Dependencies

  • jQuery
  • Bootstrap
  • jsTiles
